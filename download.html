Latest version: 2.6.2 (2018.04.21)

openSUSE: openSUSE Build Service (version 2.6.2)

Fedora: (version 2.6.2)
You can install hmtimer from copr by the following command:

dnf copr enable cges40901/hmtimer
dnf install hmtimer

Ubuntu/Mint: (version 2.6.2)
PPA for Hsiu-Ming's Timer is available at https://launchpad.net/~cges30901/+archive/ubuntu/hmtimer. You can install from the ppa by the following command: 

sudo add-apt-repository ppa:cges30901/hmtimer
sudo apt-get update
sudo apt-get install hmtimer
