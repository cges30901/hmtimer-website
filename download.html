openSUSE: openSUSE Build Service

Fedora:
You can install hmtimer from copr with the following command:

sudo dnf copr enable cges30901/app
sudo dnf install hmtimer

Ubuntu/Mint:
PPA for Hsiu-Ming's Timer is available at https://launchpad.net/~cges30901/+archive/ubuntu/app. You can install from the ppa with the following command: 

sudo add-apt-repository ppa:cges30901/app
sudo apt-get update
sudo apt-get install hmtimer
