Translation

You can help by translating Hsiu-Ming's Timer on Transifex.

To traslate Hsiu-Ming's Timer into your language, you need to have a Transifex account first. If you really don't want to use Transifex to translate, you can also download the translation file(hmtimer.ts if your language does not exist, hmtimer_XX if it already exists), translate it, and send me an e-mail with the traslated file attached. It is fine to send it to me even if you only traslate part of the ts file. If you have any questions, feel free to send me an e-mail.
